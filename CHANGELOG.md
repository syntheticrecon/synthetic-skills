# Changelog

All notable changes to this repository will be documented in this file.

The format is based on Keep a Changelog and the project uses semantic version tags.

## [Unreleased]

### Added
- Root changelog for tracking repository-level changes across releases.
- Root [AGENTS.md](AGENTS.md) and [CLAUDE.md](CLAUDE.md) instructions that require branch-based changes, changelog updates, and tag-based releases.

## [0.1.1] - 2026-03-28

### Added
- MIT license for the repository.
- Pull request template for skill and repo changes.
- Issue templates for bug reports and skill requests.
- GitHub Actions release workflow for tag-based releases.

### Changed
- Enabled branch auto-delete on merge.
- Applied branch protection to `main` with pull request reviews, conversation resolution, and linear history.
- Published `v0.1.1` as an automated GitHub release.

## [0.1.0] - 2026-03-28

### Added
- Initial public release of the Synthetic Skills repository.
- `product-design` skill for designing production-quality desktop and web apps.
- `planning-setup` skill for bootstrapping persistent planning files and repo instruction updates.
- Root repository README describing the skill catalog and repository structure.
